# Telco Customer Churn Analysis - Portfolio Project

**Mục tiêu**: Phân tích nguyên nhân khách hàng rời bỏ và đề xuất giải pháp retention cho công ty viễn thông.

## Tech Stack & Flow
- **Python**: Pandas, SQLAlchemy
- **Database**: SQL Server
- **Visualization**: Power BI (DAX)
- **Others**: Jupyter Notebook, Git

## Key Insights
- Hợp đồng Month-to-month có churn rate cao nhất.
- Khách hàng sử dụng Fiber optic và thanh toán Electronic check dễ churn.
- Tenure thấp (< 12 tháng) là nhóm rủi ro cao.
- Senior Citizen churn cao hơn nhóm còn lại.

## Business Recommendations
- Khuyến khích khách hàng chuyển sang hợp đồng dài hạn với ưu đãi.
- Cải thiện dịch vụ Fiber optic và hỗ trợ thanh toán.
- Tập trung chương trình onboarding cho khách hàng mới.
